# ar-sabaedog

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

AR Saba Dog is a project that allows you to enjoy the Saba Dog, a local delicacy from Fukui Prefecture, in Augmented Reality (AR).
